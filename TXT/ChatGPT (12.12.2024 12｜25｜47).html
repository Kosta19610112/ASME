Пример извлечения таблицы из PDF

python
import pdfplumber
import pandas as pd

# Путь к файлу PDF
file_path = 'pdf_urez.pdf'

# Открываем PDF
with pdfplumber.open(file_path) as pdf:
    # Открываем нужную страницу (например, 1)
    page = pdf.pages[0]
    
    # Извлекаем таблицу
    table = page.extract_table()
    
    # Создаем DataFrame
    df = pd.DataFrame(table[1:], columns=table[0])

# Преобразуем данные в числовой формат (если требуется)
df = df.apply(pd.to_numeric, errors='coerce')

# Выводим таблицу
print(df)

Как это работает:

  1. pdfplumber извлекает данные из PDF в табличном формате.
  2. page.extract_table() возвращает таблицу в виде списка списков, где первая строка обычно содержит заголовки.
  3. pd.DataFrame преобразует данные в DataFrame для анализа.
  4. apply(pd.to_numeric) приводит значения к числовому типу, обрабатывая текстовые или пропущенные значения.

После выполнения кода вы получите данные в виде таблицы. Если в таблице есть сложное форматирование или разделенные ячейки, возможно, потребуется дополнительная обработка.

Полностью автоматизированный подход

Вот пример автоматизации извлечения данных таблицы:

С использованием pdfplumber

python
import pdfplumber
import pandas as pd

# Путь к вашему PDF-файлу
file_path = 'pdf_urez.pdf'

# Открываем PDF-файл
with pdfplumber.open(file_path) as pdf:
    # Указываем страницу, где находится таблица
    page = pdf.pages[0]  # Замените 0 на номер страницы (начинается с 0)
    
    # Извлекаем таблицу
    table = page.extract_table()
    
    # Преобразуем таблицу в DataFrame
    if table:
        # Преобразуем таблицу в DataFrame
        df = pd.DataFrame(table[1:], columns=table[0])  # table[0] содержит заголовки
        
        # Преобразуем строки в числовой формат, если необходимо
        df = df.apply(pd.to_numeric, errors='coerce')
    else:
        print("Таблица не найдена.")

# Выводим таблицу
print(df)

Ошибка, при которой page.extract_table() возвращает None, может возникать по нескольким причинам. Вот основные из них и возможные решения:

Возможные причины и решения:

1. Неправильное форматирование таблицы в PDF

Таблица может быть не распознана как таблица из-за сложного форматирования или отсутствия четких линий, разделяющих ячейки.

Решение: Используйте параметр page.extract_table({"horizontal_strategy": "lines", "vertical_strategy": "lines"}), чтобы указать pdfplumber, искать таблицу по линиям.

Пример:

python
table = page.extract_table({
    "horizontal_strategy": "lines",
    "vertical_strategy": "lines"
})

2. Нет явных границ ячеек (линий)

Если таблица представлена в PDF только текстом (без нарисованных границ), pdfplumber может не распознать её как таблицу.

Решение: Попробуйте извлечь текст с координатами и программно разделить его.

Пример:

python
# Извлекаем текст всей страницы
text = page.extract_text()

# Анализируем текст для разделения по строкам и колонкам
print(text)  # Вывод текста для анализа

Если текст извлечен корректно, его можно преобразовать в таблицу вручную, используя правила обработки строк и разделителей.

3. Таблица находится не на указанной странице

Возможно, указанная страница (например, page = pdf.pages[0]) не содержит таблицу.

Решение: Проверьте страницы PDF-файла:

python
for i, page in enumerate(pdf.pages):
    print(f"Страница {i+1}:\n", page.extract_text())

После этого укажите нужную страницу в коде.

4. Шрифт или кодировка текста не поддерживаются

Если текст в таблице использует нестандартный шрифт или зашифрован, pdfplumber может не корректно извлечь данные.

Решение: Используйте альтернативную библиотеку, например, tabula-py, или конвертируйте PDF в более простой формат (например, Excel) через сторонние инструменты, такие как Adobe Acrobat.

5. Попробовать другой метод распознавания таблицы

Если pdfplumber не справляется, попробуйте использовать tabula-py или OCR для обработки сложных таблиц:

Пример с tabula-py:

print(df)
